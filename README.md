# Tourist App QA Testing Project

## Project Overview

This repository contains QA testing artifacts created for a mobile Tourist App developed as a university project.

The application allows users to browse tourist attractions, restaurants, resorts, and hotels in Almaty, view details, ratings, and make mock bookings.

## Testing Scope

The following modules were tested:

* Home Screen
* Top Places
* Restaurants
* Restaurant Details
* Resorts
* Resort Details
* Hotels
* Hotel Details
* Navigation
* Booking Functionality

## Testing Types

* Functional Testing
* UI Testing
* Navigation Testing
* Smoke Testing
* Negative Testing

## Deliverables

### Test Cases

* 40+ manual test cases
* Positive and negative scenarios
* Expected and actual results
* Severity and priority assessment

### Bug Reports

* Navigation defects
* UI defects
* Missing functionality
* Data display issues

## Tools Used

* Microsoft Excel
* GitHub
* iOS Simulator

## Key Findings

Several defects were identified during testing, including:

* Non-functional "See All" navigation in Top Places section
* Incorrect image mapping
* Missing reviews
* Incorrect tab labeling
* Truncated hotel names
* Placeholder text displayed in production UI
